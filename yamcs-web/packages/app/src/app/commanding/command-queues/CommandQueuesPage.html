Command Queues A command queue can be in these states:
check_circle Enabled. The queue accepts and releases commands.
pan_tool Blocked. The queue accepts commands but does not release them. These commands may be released manually, or automatically as soon as the queue moves to the enabled state.
lens Disabled. The queue does not accept commands. Any attempt to issue a command linked to a disabled queue will fail immediately.
